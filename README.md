# dcncorp-websummit2026doha
Landing page & Lead capture for Web Summit 2026 Doha

## Overview
Professional landing page for Web Summit 2026 Doha featuring:
- Responsive design optimized for all devices
- Lead capture form for registration
- Multiple sections with headings, sub-headings, and descriptive content
- Modern gradient styling and smooth animations
- Interactive navigation and form validation

## Structure
- **index.html** - Main landing page with semantic HTML structure
- **styles.css** - Professional styling with responsive design
- **script.js** - Interactive features and form handling

## Features
- Hero section with main heading and call-to-action
- About section with conference details
- Why Attend section with 4 key benefits
- Statistics section highlighting event scale
- Registration form for lead capture
- Smooth scroll navigation
- Form submission handling

## Usage
Simply open `index.html` in a web browser or serve via any web server.

For local development:
```bash
python3 -m http.server 8080
# Then visit http://localhost:8080
```
